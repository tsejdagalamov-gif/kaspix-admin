# Kaspix Admin - Управление документами и реквизитами

Полнофункциональное веб-приложение для управления документами верификации личности и реквизитами.

## 🚀 Возможности

- **Администратор (защищено паролем)**
  - Загрузка PDF документов
  - Управление документами (просмотр, скачивание, удаление)
  - Ввод и сохранение реквизитов (ФИО, номер документа, дата рождения, паспортные данные, адрес, телефон)
  - Предпросмотр реквизитов

- **Публичная страница**
  - Просмотр загруженных документов
  - Просмотр реквизитов
  - Скачивание документов

- **Мировые часы**
  - 22 часовых пояса со всего мира
  - Живое обновление времени каждую секунду
  - Возможность выбирать нужные часовые пояса
  - Красивый интерфейс с градиентом

- **Хранение данных**
  - Локальное хранилище (браузер)
  - Данные сохраняются автоматически

## 🛠️ Технологический стек

- Next.js 14
- React 18
- TypeScript
- CSS3

## 📦 Установка

```bash
# Клонирование репозитория
git clone https://github.com/tsejdagalamov-gif/kaspix-admin.git
cd kaspix-admin

# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## 🔐 Использование

### Первый вход
1. Откройте приложение
2. Введите пароль (по умолчанию: `admin123`)
3. Этот пароль будет сохранён для будущих входов

### Загрузка документов
1. Перейдите на вкладку "Документ"
2. Выберите PDF файл
3. Нажмите "Загрузить документ"

### Заполнение реквизитов
1. Перейдите на вкладку "Реквизиты"
2. Заполните поля (ФИО, номер документа, дата рождения, паспортные данные, адрес, телефон)
3. Нажмите "Сохранить реквизиты"

### Просмотр публичной страницы
- Откройте `/public` для просмотра информации без прав администратора

### Просмотр мировых часов
- Откройте `/clock` для просмотра времени в разных часовых поясах
